--------------THIS IS HOW TO RUN THE PROGRAM-----------------

---SERVERS---
1. Navigate to server folder and run the go file. (go run server.go)
2. Navigate to serverReplica folder and run the go file. (go run serverReplica.go)

---CLIENTS/NODES---
1. Navigate to node folder and run the go file with corresponding nodeID. (go run node.go A )
2. Open a new terminal
3. Repeat step 1, but use a new nodeID. (go run node.go B)
